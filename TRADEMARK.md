# Rotor Framework Trademark Policy

This document describes the policy for use of the "Rotor Framework" name, logo, and related trademarks. This policy is separate from the Apache License 2.0 that governs use of the source code.

## Overview

"Rotor Framework" is a trademark of Balazs Molnar. This trademark policy exists to:

- Protect users from confusion about what is and isn't official Rotor Framework software
- Allow the community to make fair use of the trademark for accurate, factual references
- Preserve the integrity and reputation of the project

## Permitted Uses (No Permission Required)

You may use the "Rotor Framework" name without prior permission for the following purposes:

### Factual Statements

- Stating that your application or product is "built with Rotor Framework"
- Stating that your application or product "uses Rotor Framework"
- Stating that your application or product is "compatible with Rotor Framework"
- Stating that your application or product "supports Rotor Framework"

### Community and Educational Use

- Writing articles, tutorials, or documentation about Rotor Framework
- Creating educational content that teaches how to use Rotor Framework
- Discussing Rotor Framework in presentations, videos, or podcasts
- Referencing Rotor Framework in academic papers or research

### Development and Distribution

- Distributing unmodified copies of Rotor Framework
- Distributing modified versions, provided you follow the naming guidelines below

## Naming Guidelines for Modified Versions

If you distribute a modified version of Rotor Framework, you must:

- **Do not** use "Rotor Framework" as the name of your modified version
- **Do not** use names that could be confused with "Rotor Framework" (e.g., "Rotor Framework Plus", "Rotor Framework Pro", "Rotor Framework Enterprise")
- **Do** clearly indicate that your version is a fork or derivative
- **Do** choose a distinct name for your modified version
- **You may** include a statement that your project is "based on Rotor Framework" or "derived from Rotor Framework"

### Examples

| Acceptable | Not Acceptable |
|------------|----------------|
| "MyFramework (based on Rotor Framework)" | "Rotor Framework Extended" |
| "AcmeTech UI Engine" | "Rotor Framework by AcmeTech" |
| "Built with Rotor Framework" | "Official Rotor Framework" |
| "Rotor Framework compatible" | "Rotor Framework Certified" |

## Prohibited Uses

The following uses require explicit written permission:

- Using "Rotor Framework" in a product name, company name, or domain name
- Implying official endorsement, sponsorship, or partnership with Rotor Framework
- Using "Official Rotor Framework" or similar phrasing that suggests official status
- Using the Rotor Framework name or logo in a way that suggests your product is the official Rotor Framework
- Selling products or services under the "Rotor Framework" name
- Using the Rotor Framework logo without modification
- Registering domain names containing "Rotor Framework" or confusingly similar terms

If a Rotor Framework logo exists, it may only be used:
